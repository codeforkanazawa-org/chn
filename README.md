# CfK Civic Hack Night

基本的にGitHub Pages用と、本会運営用タスク管理としてissuesを用いていきます。

# 協力していただける方

[issues](https://github.com/codeforkanazawa-org/chn/issues) にタスクがありますのでお手伝いいただける方はそこに書き込んでいただけると助かります。
